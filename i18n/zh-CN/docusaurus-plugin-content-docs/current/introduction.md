# 介绍

Apache Answer 可帮助你在用户或团队之间建立问答社区。你可以在这里分享知识并帮助用户解决问题。

主要功能：

- **提问和回答** - 提出问题并从你的用户和团队那里获得答案。
- **组织内容** - 将你的内容组织为分类和标签。
- **集成** - 与你现有的工具和服务集成。
- **游戏化** - 奖励用户的贡献。
- **现代用户界面** - 现代且响应式的用户界面。
- **可扩展** - 使用插件扩展 Apache Answer。
- **开源** - 免费且开源。

## 架构

### 前后端分离

Apache Answer 是一个单体库，包含前端和后端。
